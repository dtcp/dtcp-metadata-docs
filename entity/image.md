## Image Fields

| Name           | Type            | Optional | Description
| -------------- | --------------- | -------- | --------------------
| thumb          | array           | n        | 缩略图url,[url1,url2]
| original       | string          | n        | 原图url
| ext            | string          | n        | 后缀名称, 例 'png'
| width          | integer         | n        | 图片宽度, 1920
| height         | integer         | n        | 图片高度, 768
| size           | integer         | n        | 图片大小, 单位 byte 402269
