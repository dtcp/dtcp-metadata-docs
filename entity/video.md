## Video Fields

| Name     | Type       | Optional | Description
| -------- | ---------- | -------- | -----------
| thumb    | array(url) | n        | 简略内容url
| original | url        | n        | 原内容url,视频或者音频内容地址
| ext      | string     | n        | 内容类型,视频或者音频的内容
| size     | integer    | n        | 内容大小, 单位 byte 402269
